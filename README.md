# Python_OpenCV_TrafficLights
Python基于OpenCV的交通路口红绿灯控制系统设计

程序开发软件：PyCharm + Python3.7 + Sqlite + OpenCV

  基于python和opencv库实现一个交通路口红绿灯控制系统，主要可以自动和手动控制红绿灯，还带有视频录像功能等，主要做的具体需求如下几点：
1、三个路口显示（前台）
2、路口红绿灯显示（前台）
3、路口摄像头设置是否开启（前台）
4、路口红绿灯控制时间（前台）
5、路口红路灯直接控制（前台）
6、点击路口 显示当前摄像头内容（前台）
7、摄像头开启后录像保存，支持回放（后台）
8、当前人流统计（后台，假数据）
9、手动操作记录（后台）
10、车流人流信息变化
